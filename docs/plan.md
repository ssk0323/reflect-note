## 運用方針

- 1 マイルストーン = 1 Issue（エピック型）。Issue 内に「やること」をチェックボックスで列挙する
- Issue に紐づく PR は **複数になりうる**。PR は Issue を `Refs #N` で参照（最後の PR のみ `Closes #N`）
- マイルストーン順は基本的に上から実装する。M2 内のチェックボックス順は柔軟に組み替えてよい

### M1: feat: 認証 + データ基盤の整備

**目的**: 自分しかアクセスできないクラウド同期の土台を作る。これ以降の機能 Issue が Supabase と認証を前提に書ける状態にする。

**主な受け入れ条件**:
- [ ] Supabase プロジェクトが作成され、SDK が統合されている（`@supabase/ssr` 等、Server / Client それぞれから安全に呼べる）
- [ ] 環境変数 (`NEXT_PUBLIC_SUPABASE_URL`, `NEXT_PUBLIC_SUPABASE_ANON_KEY`, 必要なら `SUPABASE_SERVICE_ROLE_KEY`) が `.env.local` と Vercel 両方で動く
- [ ] Google OAuth ログイン画面が表示され、自分の Google アカウントでログインできる
- [ ] 自分のメールアドレス以外がログインしようとした場合は弾かれる（Supabase ポリシー / middleware / Edge Function のいずれか）
- [ ] 未ログイン状態で保護ページを開くとログインページにリダイレクトされる
- [ ] `records` テーブル（`id`, `user_id`, `type`, `answers` JSONB, `created_at`, `updated_at`）が作成され、想定インデックスが張られている
- [ ] `records` の RLS で `user_id = auth.uid()` のみ select/insert/update/delete できる
- [ ] Supabase との疎通テストが Vitest で 1 本以上ある（モック or テスト用クライアント）

**ラベル**: `type:feat`, `priority:high`

**依存**: #1 完了

**Issue 内チェックボックス（実装ステップ）**:
- [ ] Supabase プロジェクト作成（手動）+ env 設定手順を README に追記
- [ ] `@supabase/ssr` 等の依存導入と Client / Server ヘルパー実装
- [ ] Google OAuth プロバイダ設定（Supabase 側 + Google Cloud Console）
- [ ] 許可リスト方式の実装（自分のメールのみ通す）
- [ ] 認証 middleware（未ログインリダイレクト）
- [ ] `records` テーブルマイグレーション + RLS ポリシー
- [ ] 疎通テスト
- [ ] CLAUDE.md に Supabase 運用メモを追記

### M2: feat: MVP コア機能一式

**目的**: reflect-note が「使える状態」になるまでの主要機能を一気通貫で実装する。

**主な受け入れ条件**:
- [ ] 朝/夜/週目標/週振り返り/月目標/月振り返りの 6 フローで記録を作成し Supabase に保存できる
- [ ] 一問一答 UI で進捗バー・戻る・次へ・確認画面が動作する
- [ ] 入力途中で離脱しても、再開時に localStorage から下書きが復元される
- [ ] トップ画面に「今週の目標」「今月の目標」のカードと、各フロー開始のアクションカード、最新記録が表示される
- [ ] 過去記録一覧で日付グルーピング・種別ラベル・フィルタが動く
- [ ] 記録の詳細・編集・削除（確認ダイアログ）ができる
- [ ] ストリーク（朝/夜の連続記録日数）が計算・表示される
- [ ] 週次・月次の入力完了バッジ、月間達成バッジが付与・表示される
- [ ] スマホ表示でも崩れない（モック JSX のレイアウトを踏襲）
- [ ] 主要ユースケースの単体・コンポーネントテストがある

**ラベル**: `type:feat`, `priority:high`

**依存**: M1 完了

**Issue 内チェックボックス（実装ステップ・推奨順）**:
- [ ] 一問一答 UI の共通コンポーネント（progress / 戻る / 次へ / 確認画面）
- [ ] 朝のセットアップフロー（Supabase 保存まで）
- [ ] 下書き localStorage 保持
- [ ] 夜のリフレクション（時間の使い方 group 含む）
- [ ] 週の目標設定 / 振り返り
- [ ] 月の目標設定 / 振り返り
- [ ] トップ画面（今週/今月カード + アクションカード + 最新記録）
- [ ] 過去記録一覧（グルーピング + フィルタ）
- [ ] 記録の詳細・編集・削除
- [ ] streaks / achievements テーブルマイグレーション
- [ ] ストリーク計算 + 表示
- [ ] 入力完了バッジ + 月間達成バッジ

PR は **チェックボックス 1〜数件単位** で出す。レビュー可能なサイズを保つ。
